Para executar o scraper será necessário:

1) Incluir o pacote phantomjs no diretório raiz 
Link para download: http://phantomjs.org/download.html

2) Incluir geckodrive.exe (para o browser que deseja utilizar) no mesmo local
Para Mozila, ver: https://developer.mozilla.org/en-US/docs/Mozilla/QA/Marionette/WebDriver

Observação: O arquivo ids_lattes.csv contém apenas uma amostra do arquivo original, em razão do limite de armazenamento do GitHub.
Link para download do CSV: http://estatico.cnpq.br/bi/CNPQ/DadosAbertos/Lattes/IdentificadorLattes/identificador_lattes_20170708.zip
